# Portfolio

A fast, responsive personal portfolio website to showcase my projects, skills, and experience.

## Live Demo 
- **Link:** https://astro-keerthana.github.io/Keerthana-Gunaretnam/ 

## Features
- Responsive layout (mobile-first)
- Projects showcase with links (GitHub / Live)
- About + Skills section
- Contact section (email / social links)
- SEO-friendly basics (meta tags, Open Graph)
- Clean UI and smooth navigation

## Tech Stack
- **Frontend:** HTML / CSS / JavaScript *(or React / Next.js — update as needed)*
- **Styling:** Tailwind CSS / CSS Modules / Sass *(choose one)*
- **Deployment:** Vercel / Netlify / GitHub Pages

## Getting Started

### 1) Clone the repo
```bash
git clone https://github.com/astro-keerthana/Keerthana-Gunaretnam.git
cd your-repo
```

### 2) Install dependencies
> Skip if this is plain HTML/CSS/JS.

```bash
npm install
```

### 3) Run locally
```bash
npm run dev
```

Open: http://localhost:3000

## Build
```bash
npm run build
npm run start
```

## Configuration
Update these to make it yours:

- **Profile info:** name, role, bio
- **Projects:** titles, descriptions, tech, links
- **Social links:** GitHub, LinkedIn, Twitter, etc.
- **Contact:** email / form provider
